# Manifest

## Manuscript-facing code

- `code/run_all.py`: wrapper pipeline; supports `--recompute-random-baseline` and copies SI-ready outputs.
- `code/run_plos_one_analysis.py`: main analysis, schema validation, fixed 15-item operational example, location-code-based hold-out validation, calibration, subgroup checks, selective full-form escalation, order-assumed domain-balanced item sets, non-overlapping/severity-label validation, fully nested item-selection CV, retrospective response-quality audit, first-stage observable quality flags, data-quality exclusion sensitivity, and top-k bootstrap CI output.
- `code/run_tree_based_sensitivity.py`: supplementary LightGBM/XGBoost item-ranking sensitivity.
- `code/make_figures_and_summary.py`: data-derived Figure 1-Figure 5 generation, including PNG previews, 600-dpi LZW TIFFs, editable-text SVGs, and PDFs. By default, journal-upload copies are written under `<out>/figure_files_for_upload`; pass `--upload-dir figure_files_for_upload` to update the package-level upload folder.
